# Python
Data Science Examples with Python

## Overview
This repository contains a Jupyter notebook with concise Data Science and Python examples to help learn basic Python, plotting, NumPy operations, simple algorithms, data structures, and a small API call example.

## Notebooks
- Main notebook: [first_notebook.ipynb](first_notebook.ipynb)

## Contents / Highlights
- Basics: variables, lists, dictionaries, printing
- Functions: greeting example (`greet`)
- NumPy: array operations and cumulative sum (`cumulative`)
- Plotting: simple plots and a plot + table example (`table_data`)
- Data Structures: simple `Stack` class and usage
- Algorithms: `binary_search` example
- API: simple GET request using `requests`

## Requirements
- Python 3.8+
- Install required packages:
```sh
python -m venv .venv
source .venv/bin/activate   # or .venv\Scripts\activate on Windows
pip install numpy matplotlib requests jupyter
```
